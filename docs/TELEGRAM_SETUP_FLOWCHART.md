## Quick Reference

### Bot Token Format
```
Correct:   123456789:ABCdefGHIjklMNOpqrsTUVwxyz
Wrong:     123456789
Wrong:     ABCdefGHI...
Wrong:     bot123456789
```

### Chat ID Format
```
Correct:   987654321
Correct:   -1234567890  (for groups)
Wrong:     987654321.0
Wrong:     user_987654321
Wrong:     @username
```

### API URL Format
```
https://api.telegram.org/bot[TOKEN]/getUpdates
                            ^^^
                            No space!

Example:
https://api.telegram.org/bot123456789:ABC.../getUpdates
```

### Test Command
```bash
./scripts/test-notification.sh telegram
```
